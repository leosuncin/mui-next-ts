> Showcase project for how to use Material UI with Next.js

## Prerequisites

[![Greenkeeper badge](https://badges.greenkeeper.io/leosuncin/mui-next-ts.svg)](https://greenkeeper.io/)

- node >= 12
- npm >= 6

## Install

```sh
npm install
```

## Usage

**Run development server:**

```sh
npm run dev
```

**Run unit tests:**

```sh
npm test
```

**Run end-to-end tests:**

```sh
npm run test:e2e
```

**Run linter:**

```sh
npm run lint
```

**Build and run:**

```sh
npm run build
npm start
```

**Code coverage:**

```sh
npm test -- --coverage
CODE_COVERAGE=true npm run start:e2e
npx nyc report
```
